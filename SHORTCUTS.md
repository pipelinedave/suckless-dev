| Shortcut                | Description                    |
|-------------------------|--------------------------------|
| `Alt + Enter`           | Open terminal                  |
| `Alt + p`               | Launch dmenu                   |
| `Alt + j`               | Focus next window              |
| `Alt + k`               | Focus previous window          |
| `Alt + Shift + Enter`   | Bring window to master area    |
| `Alt + h`               | Decrease master area size      |
| `Alt + l`               | Increase master area size      |
| `Alt + Shift + c`       | Close focused window           |
| `Alt + Shift + q`       | Quit DWM                       |
| `Alt + [1-9]`           | Switch to workspace [1-9]      |
| `Alt + Shift + [1-9]`   | Send window to workspace [1-9] |
| `Alt + Space`           | Toggle floating layout         |
| `Alt + t`               | Tiling layout                  |
| `Alt + f`               | Floating layout                |
| `Alt + m`               | Monocle layout                 |
